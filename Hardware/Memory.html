    struct Hardware::Memory

    • Hardware::Memory
    • Struct
    • Value
    • Object

    Overview

    Memory related informations of your system.

    Most values are in kB, except HugePages ones.

    memory = Hardware::Memory.new
memory.used         # => 2731404
memory.percent.to_i # => 32
